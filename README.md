# Wage Tracker – Legal & Policy Pages

This repository hosts the legal and policy web pages for the **Wage Tracker** Android app, served via GitHub Pages.

## Pages

| Page | URL path | Description |
|---|---|---|
| Privacy Policy | `/privacy-policy/` | How the app handles user data |
| Terms of Service | `/terms-of-service/` | Rules governing use of the app |
| Data Safety | `/data-safety/` | Google Play Data Safety section details |
| Contact | `/contact/` | How to reach support |

## Structure

```
/privacy-policy/
    index.html
/terms-of-service/
    index.html
/data-safety/
    index.html
/contact/
    index.html
README.md
```

## Enabling GitHub Pages

1. Go to **Settings → Pages** in this repository.
2. Under **Source**, select the `main` branch and `/ (root)` folder.
3. Click **Save**. Your site will be available at `https://genovenzo99.github.io/wage-tracker/`.

## Local Preview

Open any `index.html` file directly in a browser — no build step required.
